# Phaser-typescript-webpack
A boilerplate for <strong>PhaserJs</strong> with:

- webpack
- typescript
- eslint
- prettier
- editor-config

## Installation

If you dont have the Node, install it [here](https://nodejs.org/en/).

Enter in the project's folder and run this.

```bash
  npm install

  # after run the code

  npm run dev
```
